Controls:<br/><br/><br/>
To control the character you need to use WASD to move.

W: To jump
A: To move to the left
S: To crouch
D: To move to the right

Spacebar: To jump and enter/exit the store
J: To make a punch

F1: To see the hiboxes of the player and the enemies.
F2: To reset the entire game (hard reset).
F3: To reset the actual level.
F4: To load the first level.
F5: To load the second level.
F6: To kill and respawn the character.
F7: To gain invincibility and is deactivated once you hit the button again.
